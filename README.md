# 🌊 CodeWave Portfolio - Uma Experiência Digital Incrível

Bem-vindo ao portfólio moderno e visualmente impressionante da **CodeWave**! Este projeto foi completamente redesenhado com foco em design contemporâneo, animações fluidas e uma experiência de usuário excepcional.

## ✨ Principais Características

### 🎨 Design Moderno
- **Tema Dark Premium**: Com gradientes sofisticados e paleta de cores moderna (violeta, azul, rosa)
- **Glassmorphism**: Efeitos de vidro fosco em cards e navbar para um look elegante
- **Gradientes Dinâmicos**: Uso inteligente de gradientes em textos, backgrounds e botões

### 🎬 Animações Suaves
- **Fade In/Out**: Animações elegantes ao carregar elementos
- **Float Effect**: Elementos flutuam suavemente na tela
- **Hover States**: Cards transformam-se interativamente ao passar o mouse
- **Scroll Animations**: Elementos se revelam conforme você rola a página
- **Twinkling Stars**: Efeito de estrelas piscantes no hero section

### 📱 Completamente Responsivo
- Design adaptável para todos os tamanhos de tela
- Menu hambúrguer para dispositivos móveis
- Grid layouts que se ajustam automaticamente
- Touch-friendly interface

### 🚀 Funcionalidades Interativas
- **Navbar Sticky**: Com blur effect que muda conforme você scrolleia
- **Smooth Scroll**: Navegação suave entre seções
- **Form Interativo**: Formulário de contato com validação
- **Mobile Menu**: Menu responsivo para dispositivos móveis
- **Botões Dinâmicos**: Efeitos visuais ao interagir

## 🎯 Seções do Portfolio

### Hero Section
A seção de boas-vindas visual com:
- Título com gradiente animado
- Subtítulo elegante
- Dois botões de CTA (Call-to-Action)
- Visual flutuante com emoji e backdrop blur
- Efeito de estrelas piscantes

### Projetos em Destaque
Grid showcase com 4 projetos destacando:
- Cards com hover effects e transformações 3D
- Ícones emoji para identificação visual rápida
- Tags de tecnologia/categorias
- Links para os projetos reais
- Efeito de brilho ao passar o mouse

### Seção Sobre
Apresentação do desenvolvedor com:
- Texto descritivo e inspirador
- Grid de 3 skills principais
- Avatar visual com animação
- Design balanceado e profissional

### Seção de Contato
Formas de entrar em contato:
- Email interativo
- Links para redes sociais
- Formulário de contato funcional
- Design acolhedor e acessível
